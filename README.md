# AtCoderSwiftCLI
![Swift](https://github.com/ry-itto/AtCoderSwiftCLI/workflows/Swift/badge.svg?branch=master)

## Usage
Run this.
```sh
atcoder-swift ABC126
```

After running above, following directories will create.
```sh
.
├── A
├── ABC.xcodeproj
├── B
├── C
├── D
├── E
└── F
```

And open `xcodeproj`.
```sh
open ABC.xcodeproj
```
